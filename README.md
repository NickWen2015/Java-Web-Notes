## JSP 技術簡介
- JSP 是 Java Server Page的簡寫，一種動態網頁技術。JSP所開發的網頁技術是基於Java的，並有跨平台的特性，並可以將業務代碼分離、組件重用、基礎的的Java Servlet功能及預編譯特性。
